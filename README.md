## 指南出处

- **全称**：《成人慢性肾脏病食养指南（2024年版）》
- **发布单位**：国家卫生健康委办公厅

## 功能特点

- **分期差异化饮食指导**：CKD 1-2期（正常蛋白预防）→ 3-5期非透析（低蛋白0.6g/kg/d）→ 透析期（高蛋白1.0-1.2g/kg/d）
- **食养原则**：覆盖CKD全分期，强调蛋白、钠钾磷的综合管理
- **中医食养**：7种证型（气虚/血虚/阴虚/阳虚/湿证/热证/瘀证）+ 7种症状食养方
- **地区食谱**：7大地区×3分期×4季节 = 84套完整食谱，含钠钾磷完整数据
- **营养数据标注**：每套食谱含能量/蛋白质/脂肪/碳水/钠/钾/磷完整标注
- **营养素速查表**：常见食物钠/钾/磷/蛋白质含量及磷/蛋白质比值
- **科普风格**：说人话、给具象量、指误区——精准但不端着

## 关键数据速查

| 项目 | 推荐量 | 通俗理解 |
|------|--------|---------|
| 蛋白质（CKD 3-5期非透析） | 0.6g/kg/天 | 60kg体重≈36g，半斤豆腐的量 |
| 蛋白质（透析期） | 1.0-1.2g/kg/天 | 60kg体重≈60-72g，需要多吃优质蛋白 |
| 钠 | <2g/天（<5g盐） | 一个啤酒瓶盖 |
| 钾（3-5期） | 视血钾水平调整 | 高钾水果蔬菜需限 |
| 磷（3-5期） | <800mg/天 | 动物内脏、坚果、可乐要避免 |
| 食盐 | <5g/天 | 一个啤酒瓶盖 |
| 烹调油 | ≤25g/天 | 两汤匙封顶 |
